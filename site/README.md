# AgendaFrame MVP

주요 언론사의 홈페이지 배치와 보도 빈도를 바탕으로 오늘의 의제를 정리하고, 같은 이슈의 언론사별 프레임 차이를 근거와 함께 비교하는 대시보드 MVP입니다.

- 공개 데모: https://agendaframe-capstone.kjh01072299206.chatgpt.site
- 상위 문서: [프로젝트 README](../README.md)

## 현재 배포 범위

- 오늘의 데이터 현황과 수집 상태
- 정책 분야별 의제 필터
- 의제 점수와 4개 구성 요소
- 언론사별 기사 수·배치·제목 비교
- 6종 프레임 비중과 근거 표현
- 관찰형 AI 리포트
- 관련 기사 정렬과 상세 정보
- 반응형·키보드 접근성·동작 줄이기 지원

현재 의제·프레임 분석 화면은 제품 검증을 위한 합성 데이터를 사용합니다. 실제 기사 메타데이터는 무단 크롤링 없이 관리자 CSV 가져오기로 별도 저장하며, 분석 파이프라인 연결 전까지 합성 결과와 구분해 표시합니다.

## 실제 데이터 가져오기

- 분석 표본: 한겨레·경향신문 / 한국일보 / 중앙일보·조선일보
- 관리자 화면: `/admin`
- CSV 양식: `/templates/agendaframe-import.csv`
- 저장 범위: 제목, 원문 URL, 매체, 섹션, 게시·수집 시각, 확인된 홈페이지 배치·순위
- 제외 범위: 기사 본문, 이미지, 댓글, 회원 정보
- 보호 방식: 배포 환경의 `IMPORT_TOKEN` 비밀값

같은 원문 URL은 한 번만 저장합니다. 배치나 순위를 직접 확인하지 않았다면 빈 값으로 두며 추정하지 않습니다. 자세한 기준은 [매체 표본과 수집 원칙](docs/source-panel.md)을 참고하세요.

## 실행

```bash
npm ci
npm run dev
npm run build
npm test
```

Sites 배포 산출물은 `npm run build`로 생성하며, 상태 확인 경로는 `/api/health`, 공개 기사 메타데이터 경로는 `/api/articles`입니다.

Node.js 22.13 이상이 필요합니다. 환경변수, API 키와 서비스 계정 파일은 저장소에 커밋하지 않습니다.
