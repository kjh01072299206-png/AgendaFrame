CSV GUIDE

필수 열과 입력 기준

source
한겨레, 경향신문, 한국일보, 중앙일보, 조선일보 중 하나
title · url
기사 제목과 해당 언론사의 HTTPS 원문 주소
published_at
ISO 8601 형식 권장: 2026-07-14T09:30:00+09:00
collected_at
관측 시각. 비워두면 업로드 시각을 사용
section
정치·경제·사회 등 선택 입력
homepage_placement
확인한 경우만 TOP, MAIN, SECTION, LIST 중 하나
homepage_rank
화면에서 확인한 노출 순서. 확인하지 않았으면 비움

한국어 열 이름 `언론사·제목·URL·게시시각·수집시각·섹션·배치·순위`도 지원합니다.
